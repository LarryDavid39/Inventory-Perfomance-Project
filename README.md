# Inventory-Perfomance-Project
Collected an inventory dataset and worked on it

### Project Overview

This project aims to provide insights into the inventory performance of a company.
By analyzing various aspect of the inventory data, we seek to identify trends, make data driven recommendations, and gain a deeper understanding of the company's perfomance.

### Tool

- Excel - Data analysis and building Dashboard


### Data Analysis

1. The company gained a stock value of 187,418,229 naira
2. The MegaBuild Ltd Supplier delivers the most
3. We calculated an average reorder level of 67.9


### Data Cleaning/Preperation

In the initial data preparation phase, we performed the folloeing task:

1. Data loading and inspection
2. Handling missing data and outliers
3. Data cleaning and formatting


### Exploratory Data Analysis

EDA involves exploring the sales data to answer key question such as:

- What is the overall stock value?
- What are the items below reorder level?
- Which supplier delivers the most?
- Which category has the most low-stock products?
- What is our stock value by category?
- 5 products with highest stock value that needs reorder?
- Are we overstocking any product?
- Which products are below the reorder level?
